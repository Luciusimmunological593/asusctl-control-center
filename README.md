## 🖥️ What this app does

asusctl-control-center is an ASUS control app for Linux. It helps you manage common laptop settings from one place. You can use it to check battery settings, fan control, lighting, and other hardware options that ASUS laptops often expose through asusctl. It can also work with supergfxctl for graphics mode control on supported systems.
